# ZayNukeButton  工具管理系统 — 使用说明

## 快速开始

在 Nuke 的 Nodes 菜单栏找到 **Z** 菜单，包含四个入口：

| 菜单项                | 快捷键    | 功能             |
| ------------------ | ------ | -------------- |
| **MyButton**       | `ALT+W | 打开工具搜索面板       |
| **Export**         | —      | 导出当前选中节点为工具    |
| **Install**        | —      | 打开工具管理界面       |
| **Refresh hotkey** | —      | 刷新快捷键（代码更新后使用） |

---

## 一、工具搜索面板（按 ALT+W 打开）

### 1.1 自动匹配

按 `ALT+W` 打开面板时，会根据当前 Nuke 中选中的节点**自动填入过滤条件**：

- 选中了 Write 节点 → 自动搜索 `C=:Write` 相关工具
- 没有选中任何节点 → 自动搜索通用工具（`NoSelectedNode`、`AnyTime`）
- 选中的节点类别多个时自动合并

### 1.2 搜索框使用

搜索框**默认只读**（暗色边框），防止误输入干扰快捷键操作。

| 操作          | 说明          |
| ----------- | ----------- |
| **Alt + E** | 激活搜索框，可输入文字 |
| **双击搜索框**   | 同上          |
| **点击其他位置**  | 搜索框自动恢复只读   |

### 1.3 搜索语法

支持多条件组合搜索，多个条件用逗号 `,` 分隔：

| 前缀    | 含义       | 示例             |
| ----- | -------- | -------------- |
| `C=:` | 按节点类名搜索  | `C=:Write`     |
| `L=:` | 按标签搜索    | `L=:合成`        |
| `P=:` | 按路径搜索    | `P=:ReadWrite` |
| `N=:` | 按名称搜索    | `N=:导入`        |
| 无前缀   | 模糊搜索全部字段 | `merge`        |

示例：

```
C=:Write, N=:导入        → 找 Write 类中名称含"导入"的工具
L=:调色, C=:Grade        → 找标签"调色"且类为 Grade 的工具
```

### 1.4 面板操作

| 操作              | 说明           |
| --------------- | ------------ |
| **Esc / ALT+W** | 关闭面板         |
| **鼠标滚轮**        | 翻页浏览工具       |
| **拖拽边框**        | 移动面板位置       |
| **仅已安装**        | 勾选后只显示已安装的工具 |
| **图标**          | 勾选后显示工具图标    |
| **⟳ 按钮**        | 清空缓存重新搜索     |

## 二、导出工具（Export）

### 2.1 使用步骤

1. 在 Nuke 中选中要导出的节点（或写好 Python 代码）
2. 菜单栏 → **Z → Export**
3. 填写信息：
   - **名字**：工具名称（必填）
   - **tip**：鼠标悬停提示
   - **适用类**：点击"添加类"选择匹配的节点类型
   - **代码**：Python 工具的代码（Nuke 工具则无需填写）
4. 点击 **Nuke工具**（导出 .nk）或 **Python工具**（导出 .py）

### 2.2 工具按钮展示规则

| 类型                    | 说明         |
| --------------------- | ---------- |
| **AnyTime**           | 任何时候都显示    |
| **NoSelectedNode**    | 未选中节点时显示   |
| **AnySelectedNode**   | 选中任意节点时显示  |
| **Write / Blur 等具体类** | 仅选中对应节点时显示 |

### 2.3 多用户共享说明

导出的工具默认只有当前用户可读写。如需**多用户共享**，请管理员在 Nuke 的 Script Editor 中执行一次：

```python
import QuanXianSet, paths_setup
QuanXianSet.setup_directory_shared_permissions(paths_setup.tools_path_get())
```

> Windows 需以管理员身份运行 Nuke，Linux/Mac 需 `sudo`。执行一次后，所有用户导出的工具自动共享。

## 三、工具管理界面（Install）

### 3.1 界面概览

| 区域          | 功能                      |
| ----------- | ----------------------- |
| **左侧列表**    | 浏览所有工具，勾选安装/收藏          |
| **详情标签**    | 查看工具 UUID、名称、标签、分类、文件路径 |
| **JSON 标签** | 查看工具元数据                 |
| **帮助标签**    | 查看工具的帮助文档               |
| **编辑标签**    | 修改工具信息                  |

### 3.2 常用操作

**安装/卸载工具**：

- 在左侧列表勾选/取消勾选安装复选框
- 点击顶部 **保存更改** 按钮

**重命名工具**：

1. 在编辑标签页输入新名称
2. 点击 **重命名工具**（会自动更新关联的 .json、.png、.md 及数据库记录）

**移动工具到其他目录**：

1. 在左侧列表选中工具
2. 编辑标签页 → 点击 **移动工具**
3. 在弹出的目录树中选择目标分类
4. 点击 **确认移动**（自动更新所有路径引用）

**修改标签/匹配类**：

1. 在编辑标签页修改内容
2. 点击 **保存标签** / **保存匹配类**

**添加图标**：

1. 将图标图片复制到剪贴板（截图或复制文件）
2. 在编辑标签页点击 **从剪贴板获取图标**
3. 预览无误后点击 **保存图标**

**管理帮助文档**：

- 点击 **📂 加载文档** 选择 Markdown 文件
- 点击 **更换文档** 替换帮助文档
- 点击 **📦 导出MD** 导出文档及图片

## 四、工具同步

当手动修改了 tools 目录下的文件（新增/删除工具），需要同步：

1. 打开工具管理界面（Install）
2. 点击顶部 **同步工具** 按钮
3. 系统会扫描 tools 目录，更新数据库

## 五、目录结构

```
MyButton/
├── tools/                  ← 所有工具存放目录
│   ├── ReadWrite/          ← 读写类工具
│   ├── Merge2/             ← 合并类工具
│   ├── color/              ← 调色类工具
│   ├── Transform/          ← 变换类工具
│   ├── 01-USER/liucx/      ← 用户自定义工具
│   └── ...
├── SQLite_file/            ← 用户数据库
├── pnghelp/                ← 帮助文档图片
└── menu.py                 ← Nuke 菜单入口
```

每个工具包含以下文件：

```
工具名.py          ← 主脚本
工具名.json        ← 元数据（自动生成）
工具名.png         ← 图标（可选）
工具名.md          ← 帮助文档（可选）
```

## 六、常见问题

**Q: 按 A 打开面板后输入框无法打字？**
A: 这是设计如此。搜索框默认只读保护，按 `Alt+E` 或双击即可激活。

**Q: 打开面板搜索框是空的？**
A: 这是正常现象。清除搜索后再次按 ALT+W 打开，会自动重新填入当前选中节点的类过滤条件。

**Q: 导出工具时报"不具备权限"？**
A: 参考 2.3 节，请管理员运行一次性目录权限配置。

**Q: 移动工具后搜索不到？**
A: 点击 Install 进入管理界面，点 **同步工具** 刷新数据库。

**Q: 代码更新后快捷键不生效？**
A: 菜单栏 → **Z → Refresh hotkey** 刷新即可。
